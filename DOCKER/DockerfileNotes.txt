## Docker Layer Architecture 

first laye : create container and add base image  to container 
             then create image from container 
             remove container 
second layer :  in second layer it takes first layer image and run cmd 
              then delete that container keeps image 

              and continueous process

FROM <imagename>                   # Base Image Name
RUN  <Cmd>                         # command line commands  
LABLE <labelname>                  # add contact if any error occuress then contact here 
ENV  <env_variable>                # set environmental variable || run container || run 'env' command to check 
                                   # usecase secret username and passwd : email send stmp python
USER <username>                    # login as new user not a root 
WORKDIR <dir>                      # login and navigate to pirticualr dir 
COPY <filepath> <container_path>   # copy 
ADD  <filepath> <container_path>   # copy file but if tar file then extract it  
                                   # if any zip link available like website then copy extracted content
CMD [<'cmd1'>]                     # create container and run cmd              
EXPOSE <port_no> expose the port
